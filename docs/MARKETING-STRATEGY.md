## TL;DR (ใจความ 30 วินาที)

1. **ตลาดนี้สินค้าเหมือนกันหมด** เพราะทุกเจ้าซื้อจากโรงงานชุดเดียวกัน → ตัวสินค้าไม่ใช่จุดขาย
2. **กับดัก:** เอา list คู่แข่งมาลงให้ครบ = กลายเป็น "OfficeMate ที่เล็กกว่า" แข่งจำนวน/ราคาแล้วแพ้
3. **ทางออก:** Curate ไม่ใช่ Replicate — คัดเฉพาะตัวเด่น นำเสนอให้ดีกว่า จัดเป็นคอลเลกชันตามโอกาส
4. **Positioning ที่ว่าง:** "พรีเมียม + ราคาโปร่งใส + ส่งเร็ว" ในตลาดที่ทุกเจ้ายัง "ขอใบเสนอราคา" และตอบช้า
5. **จุดต่างเบอร์ 1 ที่ต้องแก้ก่อน:** เว็บเราเองเปิดราคาแค่ 71/245 SKU — ติดราคาให้ครบคือ move แรก

## 1) ภาพตลาด (จากข้อมูลจริง)

| สิ่งที่เจอ | ตัวเลข | นัยเชิงกลยุทธ์ |
|---|---|---|
| ตลาด premium gift ไทย | ~THB 1.35bn โต 6–8% **กระจายตัวสูง** | ไม่มีเจ้าตลาดเด็ดขาด = ช่องว่างเปิด |
| คู่แข่ง **ซ่อนราคา** | 8,652 สินค้า เปิดราคาแค่ **1,351 (16%)** | "quote-only" = จุดอ่อนร่วมทั้งวงการ |
| ดิจิทัล/SEO อ่อน | Giftwise บล็อกหยุดนิ่ง **~19 เดือน** | ช่องเจาะด้วย content/SEO เปิดกว้าง |
| ช่อง premium/luxury | ยังว่าง | ขึ้นบนได้ ไม่ต้องสู้ราคาถูก |
| Giftwise (เจ้าใหญ่) | LINE OA **46,042 เพื่อน** · ไม่มี Lazada · ราคาซ่อนหลัง RFQ | จุดแข็ง = community; จุดอ่อน = SEO ตาย + ราคาทึบ |

**3 รายที่อันตรายสุด — อย่าสู้ตรงๆ ในจุดแข็งเขา:**
- **Giftmanufactory** — design-led, ราคาครบ 627 ตัว (Wix Stores)
- **Premium Perfect** — ทุน 10M+ มีโรงงานเอง + FB 78k
- **OfficeMate** — เครือ Central, ราคาครบ, สเกลใหญ่

## 2) ⚠️ กับดักของ "เอา list คู่แข่งมา regenerate"

ถ้าลง 8,652 SKU ให้ครบ = แข่งด้วยจำนวน+ราคา ซึ่งแพ้ OfficeMate (เครือ Central) ตั้งแต่ยังไม่เริ่ม

**เปลี่ยนกรอบคิด: Replicate → Curate**
- อย่าลงทุกตัว — เลือกเฉพาะ SKU ขายดี/มาร์จิ้นดี
- ความต่างอยู่ที่ **การคัดสรร + การนำเสนอ + การจัดเซ็ต** ไม่ใช่ขนาดแคตตาล็อก
- ใช้แกลเลอรี AI-enhanced (โปรเจกต์ image-audit 71 SKU) เป็นอาวุธพรีเซนต์

## 3) Positioning

วาง 2 แกนที่คู่แข่งทำได้แย่ แล้วยึด quadrant ที่ว่าง:

```
            PREMIUM / ดีไซน์สูง
                    │
   Giftmanufactory  │  ★ GO PREMIUM  ← ที่ว่าง: พรีเมียม + ราคาโปร่งใส
   (ดีไซน์ดี        │     (โปร่งใส + เร็ว + ดีไซน์)
    แต่ราคาทึบ)     │
  ──────────────────┼──────────────────
   คู่แข่งส่วนใหญ่    │   OfficeMate
   (ทึบ + คอมมอดิตี้) │   (ราคาโปร่ง แต่ไม่พรีเมียม)
                    │
            ราคาทึบ ←→ ราคาโปร่งใส
```

**Positioning statement:**
> **GO PREMIUM** — ของขวัญองค์กรระดับพรีเมียมที่ **เห็นราคา เห็นงานจริง สั่งได้เร็ว** ในตลาดที่ทุกเจ้ายัง "ขอใบเสนอราคา" และตอบช้า

**Brand architecture 2 ชั้น (ใช้ของที่มีอยู่แล้ว):**
- **GO PREMIUM** (core) — คัดสรร, ราคาโปร่งใส, ส่งเร็ว → HR / Marketing / SME
- **GO PREMIUM Exclusive** (Royal Navy + Champagne Gold) — bespoke / co-brand สำหรับ Super-VIP

---

## 4) จุดต่าง (Differentiation Wedges)

เจาะจุดอ่อนคู่แข่งทีละข้อ:

1. **ราคาโปร่งใส** — แสดงราคา/ช่วงราคา + MOQ ทุก SKU (คู่แข่ง 84% ซ่อน) → trust + SEO + ลด friction
   - ⚠️ GoPremium เองเปิดราคาแค่ 71/245 → **แก้ก่อนเป็นอย่างแรก**
2. **เร็ว** — ดัน `#/express` ให้ชัด ("ส่งใน X วัน") คู่แข่งตอบช้าหมด
3. **ดีไซน์/พรีเซนต์** — แกลเลอรี AI-enhanced + mockup โลโก้ลูกค้า สู้ Giftmanufactory ด้วยต้นทุนถูกกว่า
4. **Content/SEO** — Giftwise บล็อกตาย 19 เดือน; เรามี 5 บทความไทยพร้อม → wire `#/blog` เก็บ organic ที่ทุกเจ้าทิ้ง
5. **ขายเป็น solution ตามโอกาส/บุคคล** — เช่น "ชุดต้อนรับพนักงานใหม่", "ของขวัญปีใหม่ลูกค้า VIP", "Onboarding Kit" → มาร์จิ้นดีกว่า เทียบราคายากกว่า

## 5) Action Plan

### ทำทันที (สัปดาห์นี้)
1. ติดราคา/ช่วงราคา + MOQ ให้ครบทุก SKU ที่ลงเว็บ ← จุดต่างเบอร์ 1 และเป็นจุดอ่อนของเราเอง
2. คัด SKU จาก list คู่แข่ง — เอาเฉพาะตัวขายดี/มาร์จิ้นดี (อย่าลงครบ)

### 30 วัน
3. สร้าง 5–8 "คอลเลกชันตามโอกาส" จาก SKU ที่คัด
4. Wire `#/blog` เข้าเว็บ + ปล่อย 5 บทความ (เก็บ SEO ที่ Giftwise ทิ้ง)
5. ทำ `#/express` ให้มีคำมั่นเรื่องเวลาส่งชัดเจน

### 60–90 วัน
6. เปิดตัว GO PREMIUM Exclusive เป็น landing แยกสำหรับงาน bespoke
7. ระบบ RFQ ตอบเร็ว (เทมเพลต + SLA) — เปลี่ยน "ขอใบเสนอราคา" จากจุดอ่อนเป็นจุดแข็งด้วยความเร็ว

## 6) KPI ที่ควรวัด

| ด้าน | ตัวชี้วัด | เป้าเริ่มต้น |
|---|---|---|
| ราคาโปร่งใส | % SKU ที่มีราคาบนเว็บ | 71/245 → 100% |
| SEO | organic sessions/เดือน · จำนวนคีย์เวิร์ดติดหน้า 1 | ตั้ง baseline หลัง wire blog |
| Conversion | RFQ → quote → order rate · เวลาตอบ RFQ เฉลี่ย | เวลาตอบ < 4 ชม. |
| Premium mix | สัดส่วนยอดจาก Exclusive / คอลเลกชัน solution | ตั้งเป้าหลัง 90 วัน |
