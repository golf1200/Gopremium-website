SEO

ผล Audit SEO + สิ่งที่แก้แล้ว

ประเด็นความรุนแรงสถานะ
ไม่มี canonical tag — sitemap 281 URLs ชี้เข้าหน้าเดียวกัน (soft-duplicate ทั้งเว็บ)วิกฤตแก้แล้ว canonical รายหน้า + router อ่าน path จาก sitemap แล้วเรนเดอร์เนื้อหาที่ตรงจริง (/product/dw001 → หน้าสินค้านั้นจริง พร้อม title เฉพาะ)
ไม่มี OG / Twitter tags — แชร์ขึ้น LINE/Facebook ไม่มีรูป-ไม่มีคำอธิบายวิกฤตแก้แล้ว ครบชุด + og:image
ไม่มี Structured Data (JSON-LD) เลยวิกฤตแก้แล้ว Organization + WebSite ที่ head และ Product schema (ชื่อ/ราคา/รูป/SKU) ทุกหน้าสินค้า
title เดียวทั้งเว็บ ("GO PREMIUM 2.0 — มากกว่าของขวัญ...") ไม่มี keywordสูงแก้แล้ว title หลักเป็น "ของพรีเมียม ของขวัญองค์กร พิมพ์โลโก้ ครบวงจร | GO PREMIUM" + title/description เฉพาะทุก route
slug โอกาสใน sitemap (new-year, esg…) ไม่ตรง key จริงของ v2 (newyear, eco…)สูงแก้แล้ว sitemap ใช้ key จริง + ทำ alias รองรับ URL เก่า 7 ตัว
บทความ blog 2/5 เป็น orphan (ไม่มีลิงก์จากหน้าแรก)สูงแก้แล้ว หน้าแรกลิงก์ครบ 5 บทความ
5 SKU ไร้ชื่อ-ไร้ราคา (BG043, DW030, DW031, LS025, LS027) โชว์เป็นการ์ดเปล่าบนเว็บสูงแก้แล้ว ซ่อนจากทุกหน้า (master ระบุ "ยังไม่ขึ้นเว็บ" อยู่แล้ว)
catalogue.html / corporate.html เป็น orphan pagesกลางรอตัดสินใจ จะใช้เป็น landing โฆษณา (แนะนำ) หรือลบ — ถ้าใช้ ควรใส่ canonical กันชนกับหน้าหลัก
SPA hash-routing — Google ต้องเรนเดอร์ JS จึงเห็นเนื้อหา (ช้ากว่า static)กลางยอมรับได้ช่วงนี้ ระยะยาวถ้าจะจริงจัง SEO มาก ควร pre-render หน้าสินค้า/หมวดเป็น static HTML

สิ่งที่ต้องทำเองใน Google (ทำครั้งเดียว ~15 นาที)

  • Google Search Console → เพิ่ม property ทั้ง 2 โดเมน → submit sitemap.xml ใหม่
  • GA4 Admin → Events → ตั้ง generate_lead และ contact_line เป็น Key Events (ใช้วัด conversion โฆษณา)
  • ถ้าจะยิง Google Ads → ลิงก์ GA4 กับ Ads แล้ว import key events เป็น conversions
Tracking

ระบบ Tracking — เดิม "ติดแต่หูหนวก" ตอนนี้วัดครบ funnel

เดิมมีแค่ GA4 (G-JTMVQM245Y) ยิง page_view หน้าแรกครั้งเดียว — การกระทำทุกอย่างหลังจากนั้นมองไม่เห็นเลย วัด ROI โฆษณาไม่ได้

Eventยิงเมื่อใช้ทำอะไร
page_view (ทุก route)เปลี่ยนหน้าใน SPA ทุกครั้ง + โหลดแรกเห็น journey จริง: หน้าแรก → หมวด → สินค้า → ฟอร์ม
generate_lead ⭐ส่งฟอร์มขอใบเสนอราคาสำเร็จ (พร้อม occasion/จำนวน/งบ)Conversion หลัก — ผูกกับโฆษณาได้
contact_line ⭐คลิกปุ่ม LINE ทุกปุ่มทั้งเว็บ (จับอัตโนมัติ)Conversion ช่องทางหลักอันดับ 2 + รู้ว่าคลิกจากหน้าไหน
view_itemเปิดหน้าสินค้า (sku, ชื่อ, หมวด, ราคา)สินค้าไหนคนดูเยอะ → ใช้เลือกตัวทำตลาด
ai_searchใช้กล่อง AI (ทั้งหน้าแรกและหน้าแคตตาล็อก) พร้อมคำค้นเห็นว่าลูกค้าพิมพ์หาอะไร → ปรับ keyword/สต๊อกตามจริง

ยังไม่ได้ติด (ตัดสินใจเมื่อเริ่มยิงโฆษณา): Meta Pixel, LINE Tag, TikTok — โครง track() รองรับเพิ่มได้ทันที

Products & Pricing

สินค้า vs PRODUCT-MASTER.xlsx + ความสมบูรณ์ของราคา

✅ ข้อมูลตรงกัน 100%

245 SKU ตรงกันทุกตัว — ราคา ชื่อ หมวด MOQ ไม่มี mismatch เลย (0/245) · เว็บแสดง 240 หลังตัด 5 SKU ที่ master ระบุ "ยังไม่ขึ้นเว็บ"

⚠️ ราคา: ช่องโหว่ใหญ่สุดของแคตตาล็อก

⚠️ งบระดับ Executive เกือบว่าง

ราคาที่มีอยู่กระจาย: value 24 · smart 33 · premium 12 · executive แค่ 2 ตัว — filter "ผู้บริหาร/VIP" แทบไม่มีของเสนอ ควรเติมสินค้า/ราคาช่วง ฿300+

✅ รูปสินค้า

194/245 มีรูปจริง (79%) · 51 SKU ใช้ภาพ mockup รอถ่าย/หา (ตรงกับงานที่ค้างอยู่ 48+ SKU) · PRODUCT-MASTER.xlsx รีเฟรชแล้ววันนี้ ธง "มีรูป" ตรงกับเว็บแล้ว
